# C# Concepts and Exercises Solution

This solution is designed to help you learn and practice C# programming concepts through examples and exercises. It consists of two main projects: **Concepts** and **Exercises**.

## Projects Overview

### Concepts Project
The **Concepts** project contains explanations and examples of fundamental C# concepts, including:

- **Variables**: Understanding how to declare and initialize variables.
- **Data Types**: Exploring different data types available in C#.
- **Object-Oriented Programming (OOP)**:
  - **Classes**: Learning how to create and use classes.
  - **Inheritance**: Understanding how inheritance works in C#.
- **Collections**: Introduction to LINQ (Language Integrated Query) for querying collections.

### Exercises Project
The **Exercises** project provides practical exercises to reinforce your understanding of C#. The exercises include:

- **Reverse String**: A method to reverse a given string.
- **FizzBuzz**: A classic programming challenge that prints numbers with specific rules.
- **Palindrome**: A method to check if a string is a palindrome.
